Virtual Tourist

"Virtual Tourist" allows the user to pin any location in the world and save an album of photos taken at that location.

"Virtual Tourist" is an iOS application developed as part of the Udacity iOS Development Nanodegree program. It uses MapKit, URLSession networking, JSON Serialization, Reachability, the Flickr API, and UICollectionViews to allow the user to travel the world from the comfort of their iPhone.

The "Virtual Tourist" images are provided by theFlickr API.

Screenshots are provided by App Launchpad.

To download "Virtual Tourist", visit my Github repository at: VirtualTourist.
